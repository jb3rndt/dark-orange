# Changelog

All notable changes to the "dark-orange" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2026-03-27

### Added

- 3 new themes: Gunmetal Icy Blue Flat, Ink Void Neo Lime Flat, Midnight Blush Flat

### Fixed

- Update minimap background and editorGroup border

## [1.1.0] - 2026-03-26

### Fixed

- Make inactive tabs transparent and remove border

## [1.0.0] - 2026-03-26

### Added

- 5 new flat themes: Burgundy Night Flat, Charcoal Pale Ivory Flat, Crimson Nights Flat, Neon Mint Charcoal Gray Flat, Sunburst Midnight Flat
- Generate foreground colors based on the underlying accent/background

## [0.4.0] - 2024-03-12

### Changed

- Generate flat themes from a template
- Updated list and button colors and editor word highlighting

## [0.3.0] - 2023-11-30
